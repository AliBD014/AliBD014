Python Basics Guide

A beginner's guide to understanding Python programming concepts.

Syntax Variables Functions Conditionals Loops Classes

Syntax

Python programs consist of statements and indentation is crucial.

# This is a comment
print("Hello, World!")

Variables

Variables store data values in Python.

# Variable declaration
age = 25
pi = 3.14
name = "Alice"

# Print variables
print("Age:", age)
print("PI:", pi)
print("Name:", name)

Functions

Functions encapsulate reusable blocks of code in Python.

def greet(name):
    return f"Hello, {name}!"

# Call the function
print(greet("Bob"))

Conditionals

Conditionals execute different code based on conditions.

age = 18

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Loops

Loops execute a block of code multiple times.

# For loop
for i in range(1, 6):
    print("Count:", i)

# While loop
count = 1
while count <= 5:
    print("Count:", count)
    count += 1

Classes

Classes define objects with properties and methods.

class Car:
    def __init__(self, brand, year):
        self.brand = brand
        self.year = year

    def display(self):
        print(f"{self.brand} {self.year}")

# Create an object
car1 = Car("Toyota", 2020)
car1.display()

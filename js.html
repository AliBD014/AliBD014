Syntax

JavaScript syntax defines how programs are constructed.

// Single-line comment
/* Multi-line comment */

// Log output to the console

Variables

Declare variables using var, let, or const.

// Variable declaration
let name = 'Alice';
const age = 25;
var isStudent = true;

// Output variables
console.log(name, age, isStudent);

Functions

Functions encapsulate reusable blocks of code.

// Function declaration
function greet(name) {
  return `Hello, ${name}!`;
}

// Function call
console.log(greet('Bob'));

// Arrow function
const add = (a, b) => a + b;
console.log(add(2, 3));

Events

JavaScript can respond to user interactions.

// Add an event listener
const button = document.getElementById('myButton');
button.addEventListener('click', () => {
  alert('Button clicked!');
});

DOM Manipulation

Interact with and modify the Document Object Model (DOM).

// Select an element
const header = document.querySelector('h1');

// Change its text
header.textContent = 'Updated Header';

// Add a class
header.classList.add('highlight');

Asynchronous JavaScript

Handle asynchronous operations using callbacks, promises, or async/await.

// Fetch API example
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

// Async/await example
async function fetchData() {
  try {
    const response = await fetch('https://api.example.com/data');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
}
fetchData();

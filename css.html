CSS Properties Guide

Learn the fundamentals of CSS properties and how to style your webpage.

Selectors Colors Fonts Box Model Flexbox Grid

Selectors

CSS selectors are used to target HTML elements to apply styles.

/* Select all paragraphs */
p {
  color: blue;
}

/* Select by class */
.example-class {
  font-size: 16px;
}

/* Select by ID */
#example-id {
  text-align: center;
}

Colors

Colors can be specified using names, HEX, RGB, or HSL values.

/* Named color */
background-color: red;

/* HEX color */
background-color: #ff0000;

/* RGB color */
background-color: rgb(255, 0, 0);

/* HSL color */
background-color: hsl(0, 100%, 50%);

Fonts

Font styles, sizes, and families can be controlled using CSS.

/* Font family */
font-family: Arial, sans-serif;

/* Font size */
font-size: 20px;

/* Font weight */
font-weight: bold;

Box Model

The box model consists of margins, borders, padding, and the content area.

/* Box model example */
div {
  margin: 10px;
  border: 1px solid black;
  padding: 15px;
  width: 200px;
}

Flexbox

Flexbox is used to layout items in a flexible and efficient way.

/* Flex container */
.container {
  display: flex;
  justify-content: center;
  align-items: center;
}

Grid

CSS Grid Layout is a two-dimensional layout system.

/* Grid container */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 10px;
}
